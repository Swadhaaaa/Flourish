# Security Policy

## Supported Versions

We actively support security updates for the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

We take the security of our application seriously. If you discover a vulnerability, please report it to the maintainers rather than filing a public issue.

To report a vulnerability:
1. Email your findings to the repository owner or project maintainers.
2. Provide details of the vulnerability, including step-by-step instructions to reproduce it, and the potential impact.
3. We will acknowledge your report within 48 hours and work with you to coordinate a security release.

Please do not disclose security issues publicly until a patch has been published.
