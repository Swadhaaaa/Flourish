# Changelog

All notable changes to the **Flourish** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-08

### Added
- Integrated dynamic alphabet drop-and-bounce greeting for "Hello, Ayushi!" in dashboard header.
- Dual-mode workflow selection (Home Mode vs. Work Mode) with automated onboarding check-in.
- Dynamic Burnout Watch featuring AI Smart-Fill capability to auto-extract mental health metrics.
- Tone Shield module for sentiment analysis and conflicts prevention in email correspondence.
- Safe Cab module for ride-sharing coordination with interactive calendar/time pickers and carpool configurations.
- Auto Scheduler logic using Groq API for energy-aligned timeline reordering.
- Invisible Labor Tracker for logging household tasks and domestic support metrics.
- Comprehensive database orchestration supporting both local SQLite and Cloud Firestore modes.
- Production-grade code review, documentation assets, schema definitions, and workflow automations.
